\TermineGlossario{Java}
\DefinizioneGlossario{
	 Linguaggio di programmazione ad alto livello, orientato agli oggetti e a tipizzazione statica, specificamente progettato per essere il più possibile indipendente dalla piattaforma hardware di esecuzione.
}

\TermineGlossario{JavaScript}
\DefinizioneGlossario{
	Linguaggio di programmazione orientato agli oggetti  comunemente utilizzato nella programmazione web lato client per la creazione, in siti web e applicazioni web, di effetti dinamici interattivi tramite funzioni di script invocate da eventi innescati a loro volta in vari modi dall'utente sulla pagina web in uso.
}

\TermineGlossario{Jest.js}
\DefinizioneGlossario{
	Jest è un \glo{framework} di test \glo{JavaScript} costruito con particolare attenzione alla semplicità e al supporto per applicazioni web di grandi dimensioni.
}
