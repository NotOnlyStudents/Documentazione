\TermineGlossario{Walkthrough}
\DefinizioneGlossario{
	È un processo che ha l'obiettivo di rilevare la presenza di difetti attraverso una lettura critica ad ampio spettro del prodotto.
}
\TermineGlossario{Web application}
\DefinizioneGlossario{
	Con web application si intendono tutte le applicazioni distribuite via web che sono accessibili per mezzo di un network.
