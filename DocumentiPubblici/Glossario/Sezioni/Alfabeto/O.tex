\TermineGlossario{OAuth2}
\DefinizioneGlossario{
	Protocollo di rete che consente a diversi tipi di applicazioni (client) di verificare l'identità dell'utente attraverso l'autenticazione per mezzo di un server di autorizzazione e oltre a ciò di ottenere ulteriori informazioni di base sull'utente.. 
}

\TermineGlossario{Openshift}
\DefinizioneGlossario{
	Platform-as-a-Service (\glo{PaaS}) prodotto da Red Hat. È una piattaforma per applicazioni \glo{cloud} che rende semplice lo sviluppo, il \glo{deploy} e la scalabilità di applicazioni cloud. 
}

\TermineGlossario{Open source}
\DefinizioneGlossario{
	Software non protetto da copyright, segue la filosofia della produzione collaborativa e dell'accesso pubblico al codice sorgente.
}
