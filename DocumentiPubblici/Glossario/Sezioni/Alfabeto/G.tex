\section{G}
\TermineGlossario{Git}
\DefinizioneGlossario{
	Sistema di controllo versione distribuito utilizzabile da interfaccia a riga di comando. Utile per gestire progetti che siano tracciabili sia in locale che in remoto.
}
\TermineGlossario{GitFlow}
\DefinizioneGlossario{
	Modello composto da un insieme di regole predefinite per \glo{\textit{Git}} adatto alla collaborazione. 
}
\TermineGlossario{GitHub}
\DefinizioneGlossario{
	\textit{GitHub} è un servizio web di hosting per progetti software, basati sul sistema di \glo{versionamento} \glo{\textit{Git}}.
}
\TermineGlossario{Google Meet}
\DefinizioneGlossario{
	\textit{Google Meet} è un servizio di video-conferenza gratuito.
}
\TermineGlossario{GraphQL}
\DefinizioneGlossario{
	\textit{GraphQL} è un linguaggio di interrogazione lato server per interfacce di programmazione delle applicazioni, in grado di fornire ai client unicamente i dati di cui hanno bisogno. 
}
\TermineGlossario{gRPC}
\DefinizioneGlossario{
	 \glo{Framework} moderno e a prestazioni elevate che evolve il protocollo RPC (Remote Procedure Call) obsoleto. A livello di applicazione, gRPC semplifica la messaggistica tra i client e i servizi \glo{back end}. 
}
\TermineGlossario{Gulpease, Indice di}
\DefinizioneGlossario{
	Indice della facilità di lettura e di comprensione di un testo tarato sulla lingua italiana.
}
