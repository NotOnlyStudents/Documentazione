\TermineGlossario{Angular}
\DefinizioneGlossario{
	\glo{Framework} \glo{open source} per lo sviluppo di applicazioni web con \glo{licenza MIT}, evoluzione di \glo{AngularJS}. Il linguaggio di programmazione utilizzato per la scrittura di Angular è \glo{TypeScript}. Le applicazioni sviluppate in Angular vengono eseguite interamente dal web browser dopo essere state scaricate dal web server.
}
\TermineGlossario{AngularJS}
\DefinizioneGlossario{
	\glo{Framework} \glo{open source} per lo sviluppo di applicazioni web basato su \glo{JavaScript}.
}
\TermineGlossario{API}
\DefinizioneGlossario{
	Con Application Programming Interface (API) si indica un insieme di procedure per lo svolgimento di un dato compito. Tale termine spesso designa le librerie software disponibili in un linguaggio di programmazione.
}
\TermineGlossario{Asincroni}
\DefinizioneGlossario{
	I protocolli asincroni consentono la trasmissione per singolo carattere, nella trasmissione asincrona l'intervallo temporale tra il bit finale di un carattere e il bit iniziale del carattere successivo è indefinito.
}
\TermineGlossario{Async-await}
\DefinizioneGlossario{
	Approccio che permette di strutturare una funzione asincrona e non bloccante in modo simile a una normale funzione sincrona.
}
\TermineGlossario{Attività}
\DefinizioneGlossario{
	L'insieme di compiti da svolgere per attuare un processo.
}

\TermineGlossario{Auth0}
\DefinizioneGlossario{
	\glo{Identity Manager} di riferimento per il progetto.
}
\TermineGlossario{AWS}
\DefinizioneGlossario{
	Acronimo di Amazon Web Services, si tratta di un'azienda di proprietà del gruppo Amazon che offre servizi di \glo{cloud computing} su un'omonima piattaforma on demand.
}
\TermineGlossario{AWS Appsync}
\DefinizioneGlossario{
	Servizio che facilita lo sviluppo di \glo{API} GraphQL gestendo le \glo{attività} impegnative derivanti dalla connessione sicura a origini dati come \glo{AWS DynamoDB}, \glo{AWS Lambda} e altro.
}
\TermineGlossario{AWS DynamoDB}
\DefinizioneGlossario{
	Servizio di database \glo{NoSQL}. Tale DB supporta i modelli di dati di tipo documento e di tipo chiave-valore che offre prestazioni di pochi millisecondi a qualsiasi livello.
}
\TermineGlossario{AWS GameLift}
\DefinizioneGlossario{
	Soluzione di server hosting di giochi che distribuisce, gestisce e dimensiona i server \glo{cloud} per giochi multigiocatore.
}
\TermineGlossario{AWS Lambda}
\DefinizioneGlossario{
	Servizio di calcolo che esegue codice in risposta ad eventi e automaticamente gestisce le risorse richieste dal codice di programmazione.
}
